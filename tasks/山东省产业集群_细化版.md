## 一、菏泽市

### 1.1 曹县汉服产业集群

| 项目 | 数据 |
|------|------|
| **所在区县** | 菏泽市曹县 |
| **核心乡镇** | 大集镇、安蔡楼镇、阎店楼镇 |
| **企业数量** | 2282家（2024） |
| **网店数量** | 13989个 |
| **从业人员** | 近10万人 |
| **年销售额** | 72.15亿元（2023年） |
| **市场份额** | 占全国汉服40%+ |
| **设备规模** | 印花机1084台、绣花机3758台 |

#### 细化到乡镇

| 乡镇 | 产业特征 | 代表村/企业 |
|------|---------|------------|
| **大集镇** | 全国首批淘宝镇，汉服核心产区 | 李八庄村：汉服年销1亿元，马面裙占60-80% |
| **安蔡楼镇** | 完整产业链集聚 | 霍庄村：制版、布料、裁剪、印花、缝制、成衣5公里内完成 |
| **阎店楼镇** | 配套加工 | 辅料辅具、包装物流 |

#### 产业链分布

- **制版**：安蔡楼镇霍庄村
- **布料**：大集镇、安蔡楼镇
- **裁剪**：大集镇李八庄村
- **印花绣花**：大集镇（0.8-1元/万针，成本优势）
- **缝制成衣**：大集镇、安蔡楼镇
- **销售物流**：全镇覆盖

#### 增长趋势
- 2023年马面裙拜年服销售额超3亿元
- 2024年上半年网上销售额40.6亿元，增长18%
- 2023年以来新增线下体验店124家

## 二、潍坊市

### 2.1 寒亭区/坊子区风筝产业集群

| 项目 | 数据 |
|------|------|
| **所在区县** | 潍坊市坊子区（核心）、寒亭区、奎文区 |
| **企业数量** | 600-700家市场主体 |
| **从业人员** | 8万余人 |
| **年销售额** | 20多亿元 |
| **市场份额** | 国内80%+，国际85%+ |
| **出口企业** | 39家有自主出口权 |
| **出口地区** | 欧美、东南亚50+国家 |
| **电商占比** | 约50% |

#### 细化到乡镇/村

| 乡镇/村 | 产业特征 | 数据 |
|---------|---------|------|
| **坊子区王家庄子村** | "中国风筝产业第一村" | 200+企业，4700人口，50%从业，年销近亿只 |
| **坊子区三利风筝厂** | 电商标杆 | 日电商订单1000+单，年销200万只，600万+ |
| **寒亭区** | 传统风筝核心区 | 潍坊风筝会举办地 |
| **奎文区** | 风筝产业集聚 | 风筝企业集中分布 |

#### 王家庄子村详情

- **位置**：坊子工业发展区
- **企业**：200多家风筝企业
- **从业人员**：全村4700人，超过一半从事风筝产业
- **产品**：蝴蝶、沙燕、龙头蜈蚣等，大的几十米，小的巴掌大
- **创新**：声光电动态风筝、3D风筝
- **电商**：80%企业有网店，电商直播年销近亿元
- **出口**：欧美、东南亚50+国家，年销近亿只
- **收入**：村民人均年收入3.5万元，村集体收入30多万

## 产业痛点总结

| 痛点 | 表现 |
|------|------|
| **人才断层** | 熟练手工艺人老龄化，年轻人不愿学 |
| **机械化难** | 手工制作占比高，难以规模化 |
| **品牌化弱** | 多为OEM代工，自主品牌少 |
| **同质化** | 低价竞争严重 |
| **原材料** | 天然材料供应受限，价格波动 |
