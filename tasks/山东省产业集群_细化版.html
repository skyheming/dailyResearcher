    产业痛点总结

    | 痛点 | 表现 |

    |------|------|

    | **人才断层** | 熟练手工艺人老龄化，年轻人不愿学 |

    | **机械化难** | 手工制作占比高，难以规模化 |

    | **品牌化弱** | 多为OEM代工，自主品牌少 |

    | **同质化** | 低价竞争严重 |

    | **原材料** | 天然材料供应受限，价格波动 |
